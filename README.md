# Students Management Table

A frontend CRUD application built with **React, TypeScript, TailwindCSS, and Vite**.

## 🚀 Live Demo

https://your-vercel-link.vercel.app

## 📦 GitHub Repository

https://github.com/yourusername/students-management-table

## ✨ Features

* View students in a table
* Add new students with validation
* Edit student details with pre-filled form
* Delete students with confirmation dialog
* Search students by name or email
* Excel export for full or filtered data
* Simulated loading state
* Data persistence using LocalStorage

## 🛠 Tech Stack

* React
* TypeScript
* Vite
* TailwindCSS
* React Hook Form
* Yup Validation
* XLSX (Excel export)

src

## ⚙️ Setup

Clone the repository:

```
git clone https://github.com/yourusername/students-management-table.git
```

Install dependencies:

```
npm install
```

Run locally:

```
npm run dev
```
